# Changelog

All notable changes to this project are documented here, following
[Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.2]

### Changed
- Migrated icons to [astro-iconset](https://www.npmjs.com/package/astro-iconset)
- Upgraded Astro to 7.0.6
- Upgraded Tailwind CSS and `@tailwindcss/vite` to 4.3.2
- Upgraded React and React DOM to 19.2.7
- Bumped `@types/react` to 19.2.17

## [0.0.1]

### Added
- Initial release of AstroFlow template
- Home page with hero, features, stats, and testimonials
- Capabilities page showcasing services
- Use Cases page with industry-specific solutions
- Facilities page with location information
- Request Quote (RFQ) form page
- Documentation page with resources
- Responsive design for all screen sizes
- React components for interactivity
- Tailwind CSS 4 for styling
- TypeScript support
- Smooth animations with Motion library
